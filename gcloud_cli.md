**What is gcloud CLI**

gcloud CLI (Google Cloud Command Line Interface) is the command-line tool used to create, manage, and interact with Google Cloud resources from a terminal.

Instead of clicking in the console, you run commands.

**What can you do with gcloud**

Using gcloud, you can:

- Create & manage VMs (Compute Engine)  
- Deploy apps to App Engine  
- Manage GKE clusters  
- Configure IAM & service accounts  
- Work with Cloud Storage  
- Set projects, regions, and authentication

**Install**

https://docs.cloud.google.com/sdk/docs/install-sdk

**Check gcloud installation**

    gcloud version

**Login**

    gcloud auth login

**Set project** 

    gcloud config set project my-project-id

**List VMs**

    gcloud compute instances list

**Manage GKE**

    gcloud container clusters list

**Cloud Storage**

     gcloud storage buckets list
